# 🇺🇦 Монополія України

Веб-гра «Капіталіст України» (Монополія) на чистому HTML + CSS + JavaScript. Працює одразу в браузері — без серверів і без встановлення додатків.

## Що вже є

- 40 клітинок з українськими містами (Київ, Львів, Одеса, Харків, Дніпро, Запоріжжя, Полтава)
- 4 залізниці / аеропорти, 2 порти, 2 податкові клітинки
- Картки **Шанс** і **Екскурсія** (32 події)
- Купівля, оренда, монополії, будинки 🏠 та готелі 🏨
- В'язниця 🔒 з усіма правилами (дублі, картка, штраф 50)
- Застава нерухомості та викуп
- Банкрутство та визначення переможця
- Підтримка від 2 до 6 гравців (по черзі на одному комп'ютері)

## ▶️ Як запустити локально (на вашому комп'ютері)

### Найпростіший спосіб
1. Відкрийте папку **monopolia** на вашому Робочому столі
2. Подвійним кліком відкрийте `index.html` — гра запуститься у вашому браузері (Chrome, Edge, Firefox, Safari)

### Через Visual Studio Code (рекомендовано — потім легше деплоїти)

1. Відкрийте VS Code
2. **File → Open Folder** і виберіть папку `monopolia`
3. Встановіть розширення **Live Server**:
   - Натисніть на іконку розширень зліва (квадратики)
   - Знайдіть «Live Server» (автор: Ritwick Dey)
   - Натисніть **Install**
4. Відкрийте файл `index.html`, клікніть правою кнопкою → **Open with Live Server**
5. Гра відкриється у браузері за адресою `http://127.0.0.1:5500`

Тепер будь-які зміни у файлах одразу оновлюються в браузері — це найзручніший спосіб розробки.

## 🌐 Як викласти гру в інтернет (БЕЗКОШТОВНО)

Найпростіший спосіб — **GitHub Pages**. Це безкоштовний хостинг для статичних сайтів від GitHub.

### Крок 1: Створіть обліковий запис GitHub
1. Перейдіть на [github.com](https://github.com)
2. Натисніть **Sign up** і зареєструйтесь (потрібен лише email)

### Крок 2: Завантажте папку на GitHub
**Найпростіше — через сайт без термінала:**
1. На GitHub натисніть **+** → **New repository**
2. Назва: `monopolia` (або як хочете)
3. Постав галочку **Public**, натисніть **Create repository**
4. На наступній сторінці натисніть посилання **uploading an existing file**
5. Перетягніть **усі файли** з папки `monopolia` (index.html, style.css, script.js, README.md) у вікно браузера
6. Внизу натисніть **Commit changes**

### Крок 3: Увімкніть GitHub Pages
1. У вашому репозиторії перейдіть у **Settings** (вгорі праворуч)
2. У лівому меню оберіть **Pages**
3. У розділі **Source** виберіть **Deploy from a branch**
4. Виберіть гілку **main** і папку **/ (root)**, натисніть **Save**
5. Через 1-2 хвилини оновіть сторінку — з'явиться зелене посилання типу `https://ВАШ-ЛОГІН.github.io/monopolia/`
6. Це і є ваша гра в інтернеті! Поділіться посиланням з друзями.

## 🌍 Про справжній онлайн-мультиплеєр

Зараз гра працює в режимі «по черзі на одному пристрої» (як справжня настільна гра). Щоб гравці могли грати з різних комп'ютерів у реальному часі, потрібен **сервер** — це наступний (значно складніший) крок:

### Варіанти для майбутнього розвитку:
- **Firebase Realtime Database** — безкоштовний сервіс від Google, ідеально для початківців. Гравці підключаються через браузер, стан гри синхронізується автоматично.
- **Supabase** — альтернатива Firebase з відкритим кодом
- **Власний Node.js + WebSocket** сервер на безкоштовному хостингу типу Railway або Render

Коли захочете додати онлайн — можемо переробити логіку так, щоб стан зберігався в реальному часі. Найкраще починати з **Firebase**.

## 🎮 Правила гри (коротко)

- **Старт:** кожен гравець має ₴1500
- **Хід:** кидаєте 2 кубики, рухаєтесь стільки клітинок
- **Купівля:** на вільній ділянці можете купити її
- **Оренда:** на чужій — платите оренду власнику
- **Монополія:** володіти всією кольоровою групою → оренда ×2 і можна будувати
- **Будинки:** 🏠 збільшують оренду, по 4 на кожній → готель 🏨
- **СТАРТ:** проходите → +₴200
- **В'язниця:** заплатити ₴50 / викинути дубль / використати картку
- **Перемога:** залишитись єдиним не-банкрутом

Усі деталі — у файлі `kapitalist-ukraina.pdf`.

## 🛠️ Налаштування / редагування

Хочете змінити назви вулиць, ціни, картки? Усі дані гри знаходяться у верхній частині файлу `script.js`:

- Масив `BOARD` — клітинки дошки (40 шт.)
- Масив `CHANCE_CARDS` — картки «Шанс»
- Масив `EXCURSION_CARDS` — картки «Екскурсія»
- Константа `TOKEN_COLORS` / `TOKEN_ICONS` — фішки гравців

Просто змініть значення і збережіть файл — зміни одразу побачите у браузері.
